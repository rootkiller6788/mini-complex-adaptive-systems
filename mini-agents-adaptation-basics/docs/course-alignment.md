## Key Course Intersections

The intersection of all nine curricula converges on these core topics (all implemented):

1. **Agent abstraction** (all 9 curricula)
2. **MDP formulation** (8/9 curricula)
3. **Q-learning** (7/9 curricula)
4. **Game theory fundamentals** (7/9 curricula)
5. **Multi-armed bandits** (6/9 curricula)
6. **Nash equilibrium** (6/9 curricula)
7. **Swarm/evolutionary methods** (5/9 curricula)
