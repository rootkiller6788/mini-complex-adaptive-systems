# Course Tree: NK Fitness Landscapes

## Prerequisites

This module depends on knowledge from:

```
Probability Theory
  └── Random variables, expectation, correlation
      └── NK Landscape Statistics

Combinatorics
  └── Boolean hypercube, Hamming distance, bit operations
      └── Genotype representation and manipulation

Algorithms
  └── Local search, hill-climbing, simulated annealing
      └── Adaptive Walk Strategies

Game Theory
  └── Nash equilibrium, best-response dynamics
      └── NKCS Coevolution

Discrete Fourier Analysis
  └── Walsh-Hadamard transform on {0,1}^N
      └── Epistasis Spectral Decomposition

Evolutionary Biology
  └── Wright's fitness landscapes, Kauffman's NK model
      └── Core NK Model

Computational Complexity
  └── NP-hardness, phase transitions
      └── NK Global Optimum Hardness

Statistical Physics
  └── Spin glasses, rugged energy landscapes
      └── NK-Spin Glass Correspondence
```
